# Somewhere Between Timelines

A private, mobile-first birthday archive for Lia, created by Quin.

The experience follows one shared timeline from May 7 to Lia’s birthday on July 22. It combines letters, drawings, photographs, Beomi, distance, private jokes, and memories made together.

## Privacy

The site opens with a personal secret-word gate. This is a gentle privacy boundary, not secure authentication: a static website cannot fully protect files from someone technically determined to retrieve them. Personal media should therefore remain intentionally selected, cropped, and reviewed before publication.

## Local preview

1. Install dependencies with `npm install`.
2. Start the site with `npm run dev`.
3. Open `http://localhost:3000`.

## Publishing

GitHub Actions exports the static site and publishes it through GitHub Pages. Repository visibility and Pages visibility should be reviewed before enabling the workflow.
